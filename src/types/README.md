# Types Directory

This directory contains all TypeScript type definitions and interfaces.

## Type Categories:
- **Message Types**: MessageObject, MessageType, Priority
- **Workflow Types**: WorkflowState, Task, TaskStatus, WorkflowStatus
- **Agent Types**: Agent, AgentState, AgentStatus
- **Error Types**: ErrorContext, ErrorType, RetryPolicy
- **Domain Types**: Intent, FAQEntry, Citation, Document, etc.
