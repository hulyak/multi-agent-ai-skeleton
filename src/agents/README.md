# Agents Directory

This directory contains all agent implementations for the multi-agent system.

## Structure

Each agent should:
- Extend the `BaseAgent` abstract class
- Implement the `Agent` interface
- Handle specific message types
- Maintain its own state

## Agents to be implemented:
- Intent Detection Agent
- FAQ Agent
- Escalation Agent
- Retrieval Agent
- Summarization Agent
- Citation Agent
